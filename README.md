# Test Task 1
Программа ищёт в списке занятых IPv4 адресов наименьший свободный, не заканчивающийся на 0.

Принцип работы: 
1. Каждый IP адрес конвертируются в числовое представление и заносится в массив.

2. Полученный массив сортируется.

3. В массиве последовательно проверяются пары соседних элементов.

4. При обнаружении свободного адреса (разрыва) между ними, если он не заканчивается на 0, то найденный адрес возвращается.


# Запуск
Для запуска склонируйте репозиторий или скачайте его как архив и распакуйте в папку.


Затем перейдите в директорию проекта и запустите его командой:
```
cd first
dotnet run
```

Входные данные считываются из файла `data.txt`.
